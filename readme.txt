== Description ==
Use the [bw_dash] shortcode to display icons from

* dashicons - the official icon font of the WordPress admin
* genericons - the genericons font, as used by Jetpack
* texticons - text icons using the same method as for dash icons.


== Installation ==
1. Upload the contents of the oik-cons plugin to the `/wp-content/plugins/oik-cons' directory
1. Activate the oik-cons plugin through the 'Plugins' menu in WordPress
